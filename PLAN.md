# PolApex · 政治登顶 开发方案

> Apex 家族政治版，独立项目目录 `trae_projects/polapex`，bundle id `com.polapex.app`。
> 参考上级 `APEX_APP_开发范式.md`：复用家族骨架，但必须按政治学科特性「换心脏」。

## 一、定位三件套

**一句话**：把初中道法和高中思想政治从「背了就忘、会背不会答」变成一套可检索、可迁移、可输出的高分系统。

- **命名**：PolApex · 政治登顶。
- **主色**：法治青蓝 + 红金强调色。青蓝代表理性、秩序、法治；红金用于国家制度、主线进度和高权重考点。
- **主战场**：高考思想政治，资源占 80%。
- **入门层**：初中道德与法治，建立法治、国情、责任、集体、生命、规则等基础概念。
- **冲刺层**：高考主观题、时政材料题、选择题陷阱、选必综合。

## 二、灵魂判断：政治不是背诵 App 换皮

政治学科理论性强、死记硬背多，但高分不来自「背得多」，而来自四件事：

1. **背得准**：教材术语、核心概念、主体职责、原理边界不能混。
2. **调得出**：看到材料和设问，能迅速定位模块、主体、知识点。
3. **写得像答案**：不是泛泛议论，而是「原理句 + 材料句 + 结论句」。
4. **避得开坑**：选择题大量错在主体错配、因果倒置、范围越界、绝对化、无中生有。

由此推导 PolApex 的 4 个杀手模块：

| 灵魂 | 杀手模块 |
|---|---|
| 理论多、原话多 | **高权重记忆引擎**：原话卡、边界卡、主体卡、场景卡、模板卡 |
| 材料题难迁移 | **材料切片机**：主体、行为、对象、目标、知识点自动拆分 |
| 主观题不会写 | **答案工厂**：审设问 → 定模块 → 定主体 → 调知识 → 贴材料 → 成答案 |
| 选择题陷阱密集 | **秒杀武器库**：主体错配、绝对词、因果链、范围锁定、正确无关排雷 |

## 三、产品内核哲学

**不做题海，做「知识点全覆盖 + 高权重强记忆 + 答案输出闭环」。**

每个知识点走闭环：

```
学：教材术语 + 概念边界
记：原话卡 / 边界卡 / 主体卡 / 场景卡 / 模板卡
练：选择题排雷 / 材料切片 / 主观题短答
战：Boss 双解对决，常规硬背 vs 武器秒杀
复：按权重进入智能复习和错题诊断
```

验收指标不是刷题数，而是：

- 知识点覆盖率
- S 级掌握率
- 主体错配率
- 材料命中率
- 术语准确率
- 主观题要点完整率

## 四、知识点权重模型

每个知识点必须有权重，App 的学习顺序、复习频率、题量分配、首页推荐都由权重驱动。

```
importance =
  0.35 * 高考频率
+ 0.25 * 课程核心地位
+ 0.20 * 跨模块迁移度
+ 0.15 * 区分度
+ 0.05 * 记忆易错度
```

| 等级 | 分数 | 产品动作 |
|---|---:|---|
| S | 90-100 | 主线必经、每日复习、主观题模板、秒杀战例、错题重点追踪 |
| A | 75-89 | 主线重点、章节 Boss、选择题陷阱训练 |
| B | 55-74 | 图谱覆盖、常规练习、错题触发复习 |
| C | 0-54 | 基础识别、查漏补缺、考前浏览 |

## 五、政治专属记忆范式

PolApex 的记忆系统不是普通闪卡，而是按政治提分场景拆成 5 类：

| 卡片类型 | 训练目标 | 示例 |
|---|---|---|
| 原话卡 | 背准教材高频表述 | 中国共产党的领导是中国特色社会主义最本质的特征 |
| 边界卡 | 防止概念混淆 | 政府依法行政 vs 党依法执政 |
| 主体卡 | 解决主体错配 | 党、人大、政府、政协、法院、检察院、公民、企业、市场分别能做什么 |
| 场景卡 | 从材料召回知识点 | 平台经济、反垄断、价格监管 → 市场调节 + 宏观调控 |
| 模板卡 | 主观题输出 | 为什么 = 必要性 + 重要性 + 意义；如何做 = 主体 + 措施 + 结果 |

复习层级：

1. **认得出**：选择题中识别正确表述。
2. **背得出**：遮住关键词默写。
3. **分得清**：给两个相近概念做边界判断。
4. **用得上**：给材料片段，写出可得分答案句。

按权重调整 SM-2 间隔：

- S 级：1/2/4/7/15/30 天，错一次回 1 天。
- A 级：1/3/7/15/30/60 天。
- B 级：2/7/15/30/60 天。
- C 级：7/30/60 天。

## 七、灵魂模块

### 1. 概念星图

类似 ChemApex 的元素星图，但政治版是知识网络图：

- 节点：知识点、主体、原理、关键词、易混概念。
- 连线：包含、对比、因果、主体职责、材料触发。
- 热度：按权重、错题率、近 7 天复习状态着色。

示例：

```
政府 -> 依法行政 / 履行职能 / 建设服务型政府
人大 -> 立法权 / 决定权 / 任免权 / 监督权
党 -> 总揽全局 / 协调各方 / 依法执政
政协 -> 政治协商 / 民主监督 / 参政议政
```

### 2. 材料切片机

把一段材料拆成：

1. **主体**：党、政府、人大、企业、公民、市场、社会组织。
2. **行为**：出台政策、监督、创新、分配、协商、执法、维权。
3. **对象**：经济、法治、文化、生态、民生、国际关系。
4. **目标**：效率、公平、民主、法治、安全、发展、共同富裕。
5. **可召回知识点**：自动匹配 S/A 级知识点。

### 3. 答案工厂

主观题输出统一训练成流水线：

```
审设问 -> 定模块 -> 定主体 -> 召回原理 -> 切材料 -> 写答案句 -> 查遗漏
```

每个答案句要求三件套：

```
教材术语 + 材料对应 + 结果/意义
```

### 4. 易混概念裁判所

专门解决政治学科「看着都对」的问题：

- 党领导立法、保证执法、支持司法、带头守法，不等于党直接行使国家权力。
- 政府履行职能，不等于包办一切；市场在资源配置中起决定性作用。
- 人大是国家权力机关，政协不是国家机关。
- 公民有监督权，但不能直接行使国家权力。
- 法院行使审判权，检察院行使检察权，监察委员会行使监察权。
- 文化有继承也有创新，不能只讲传统不讲时代。

## 九、秒杀解法

### 选择题秒杀

| 秒杀法 | 识局信号 | 操作 |
|---|---|---|
| 主体错配秒杀 | 选项出现党/政府/人大/政协/公民/司法机关 | 看主体有没有资格做这个动作 |
| 绝对词排雷 | 根本、唯一、彻底、消除、一切、直接 | 先警惕，再回教材边界验证 |
| 因果链秒杀 | 因此、从而、必然导致 | 检查中间环节是否缺失或倒置 |
| 范围锁定 | 设问限定经济/政治/哲学/文化 | 超出模块即使正确也不选 |
| 材料贴合 | 选项正确但材料没说 | 正确无关直接排 |
| 哲学定位 | 设问问哲学道理 | 先分唯物论/认识论/辩证法/历史唯物主义，再选原理 |

### 主观题秒杀

政治主观题不存在真正「不背就秒」，所谓秒杀是把答题路径压缩成固定流程。

材料题六步法：

1. 圈设问：为什么 / 怎么做 / 体现 / 说明 / 意义 / 评析。
2. 定范围：经济 / 政治 / 哲学 / 文化 / 法律 / 国际。
3. 定主体：党 / 国家 / 政府 / 人大 / 公民 / 企业 / 市场 / 社会。
4. 调知识：优先调 S 级知识点，再补 A 级。
5. 贴材料：每个要点必须有材料触发词。
6. 成答案：教材术语在前，材料解释在后，意义收尾。

高分句式：

```
坚持 / 发挥 / 完善 / 推动 + 教材术语，
有利于 + 材料目标 / 现实意义，
从而 + 更高层结果。
```

## 十、题库与覆盖策略

政治要「尽量覆盖每一个知识点」，不能只堆题。题库按知识点反向生成：

```
KnowledgePoint
  -> 1 张原话卡
  -> 1 张边界卡
  -> 1 道基础选择题
  -> 1 道陷阱选择题
  -> 1 个材料触发片段
  -> S/A 级再追加 1 道主观题或 Boss 对决
```

最低覆盖标准：

- S 级：至少 12 个训练项，含主观题模板、双解对决、错因复盘。
- A 级：至少 8 个训练项，含陷阱选择题和材料短答。
- B 级：至少 4 个训练项，覆盖识别、理解、基础应用。
- C 级：至少 2 个训练项，保证不漏点。

## 十三、诚信红线

- 不臆造「某年某卷某题」出处。
- 不把政治学习做成空泛鸡汤，所有答案必须回到教材概念和设问要求。

## 十六、分阶段路线

每阶段必须独立可构建、可验收。

- **P0 骨架**：project.yml + 主题/设计系统 + App 入口 + 4 Tab 空壳 + 图标。
- **P1 脊梁**：主线 28 关数据模型 + 权重模型 + 进度管理 + 内购划线 + 付费墙 + PromoView。
- **P2 记忆引擎**：五类记忆卡 + 加权 SM-2 + 高权重复习队列 + 术语准确率。
- **P3 概念星图**：主体职责图、知识点图、易混概念图、权重热度。
- **P4 答案工厂**：材料切片机、主观题六步法、答案句模板、要点诊断。
- **P5 武器库**：选择题排雷、主体定位、哲学定位、经济传导链、Boss 双解对决。
- **P6 内容全覆盖**：初中 + 高中必修 + 选必知识点、题库、模板卡补齐。
- **P7 情感钩子**：思想家/法治人物/制度故事档案馆，篇末钩回考点。
- **P8 收尾上线**：iPad 适配、图标压平、StoreKit、docs、上线准备包、xcodebuild test 全过。

## 十七、一期 MVP

一期先做能直接提分的闭环：

1. 主线 28 关框架，先填满 S/A 级知识点。
2. 武器库 12 把：主体定位、设问翻译、材料切片、经济传导链、哲学定位、矛盾三问、法治四层、选择题五排雷、文化三维、国际关系、热点转译、主观题六步法。
3. 概念星图 MVP：高中必修四本 + 初中核心法治/国情。
4. 记忆卡：每个 S/A 点至少原话卡、边界卡、场景卡。
5. Boss 对决 20 题：选择题 12、主观题 8。
6. 错题诊断：主体错配、范围越界、因果倒置、术语不准、材料没贴。

**一句话验收标准**：学生打开 App 不是「再背一遍政治」，而是每天知道自己最该背什么、为什么错、下次遇到材料该用哪把武器、答案应该长什么样。
